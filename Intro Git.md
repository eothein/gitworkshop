# Work with SSH keys

1. Make sure Git Bash is installed
2. Create an SSH key pair (`ssh-keygen`)
3. Register public key (`~/ssh/id_rsa.pub`) on Github

<https://help.github.com/articles/adding-a-new-ssh-key-to-your-github-account/>
